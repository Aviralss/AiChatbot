# 🤖 AI Chatbot with Python & Streamlit

A simple AI-powered chatbot built using **Python**, **Streamlit**, and the **Google Gemini API**. The application provides a clean chat interface where users can interact with Google's Gemini model and receive AI-generated responses in real time.

This project was built as part of my journey into AI application development, helping me understand API integration, prompt handling, and debugging real-world issues.

---

## 🚀 Features

- 💬 Interactive chat interface
- ⚡ AI-generated responses using Google Gemini
- 🖥️ Built with Streamlit
- 🎨 Clean and minimal user interface
- 🔄 Real-time conversation experience

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AiChatbot.git
```

### 2. Navigate to the project

```bash
cd AiChatbot
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

You can generate an API key from **Google AI Studio**.

### 5. Run the application

```bash
streamlit run chatbot.py
```

Some of the issues encountered during development included:

- 503 Service Unavailable errors
- API quota (429) errors
- Empty responses despite no visible errors
- Incorrect response parsing
- Model configuration issues

Debugging these problems helped me better understand how AI APIs work and reinforced the importance of testing assumptions step by step.
